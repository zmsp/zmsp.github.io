**Scoreboard** is a professional real-time tool designed to keep the game moving. While it started as a standalone project, it is now a core feature of [Team Maker Buddy](https://teambuddy.shahadat.us/), providing referees and players with the smartest way to manage match progress.

### Features
- **Multi-Sport Support**: Ready-to-use presets for popular sports like Volleyball, Basketball, and Soccer.
- **Real-Time Tools**: Built-in half-time trackers and customizable timers to maintain a professional game flow.
- **Customizable Rules**: Easily adjust game durations and scoring increments on the fly.

## Tech Stack
- **Framework**: Flutter
- **Storage**: `hive` (Local NoSQL)
- **Integration**: Integrated into the Team Maker Buddy ecosystem.
