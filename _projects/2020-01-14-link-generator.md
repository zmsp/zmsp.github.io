Create shareable social network and messaging links for WhatsApp, Facebook, X/Twitter, LinkedIn, Instagram, TikTok, and more!

## Features

- **App Deep Linking**: Prefer native app launching when available, falling back to HTTPS.
- **Universal Link Generation**: Create direct message links and profile links for popular platforms.
- **QR Code Support**: Generate QR codes for any link with a single tap.
- **Recent History**: Keep track of your generated links and clear them anytime.
- **Auto-Save**: State is preserved across reloads.
- **Cross-Platform**: Built with Flutter, works seamlessly on Web, Android, and iOS.
