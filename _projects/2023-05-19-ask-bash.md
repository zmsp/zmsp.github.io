**ask** is a powerful CLI tool that bridges the gap between natural language and the terminal. It allows you to generate, refine, and execute complex shell commands safely using AI.

{: .notice--warning}
**Safety First:** Unlike many AI CLI tools, `ask` includes a **Dangerous Command Guard** that warns you before executing high-risk operations like `rm -rf /` or `sudo`.

## Key Features

- **Command Generation**: Get precise shell commands from plain English descriptions.
- **Context Awareness**: Pipe content for context (e.g., `cat file | ask "summarize this log"`).
- **Git Integration**: Generate intelligent commit messages based on your staged changes.
- **Provider Choice**: Seamlessly switch between OpenAI and Google Gemini.

## Usage Example

```bash
ask "list all .log files modified in the last 7 days"
# Suggested: find . -type f -name "*.log" -mtime -7
```

## Tech Stack

- **Languages**: Bash (Linux/macOS), PowerShell (Windows)
- **AI Models**: OpenAI GPT-4, Google Gemini
- **Integrations**: GitHub, Git
