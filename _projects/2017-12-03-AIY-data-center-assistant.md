I built this using the Google AIY Voice Kit and a Raspberry Pi. It’s a custom voice assistant that I programmed to handle data center tasks—like checking IP addresses or performing `nslookup`—without needing to type anything.

## The Build
The kit came as a cardboard box with a speaker and microphone. After assembling the hardware, I trained the software to recognize my voice and integrated it with Slack. Now, I can just ask it to read or send messages while I'm working on other things.
